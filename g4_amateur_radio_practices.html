Module 4: Amateur Radio Practices

Subelement G4 — Station Setup, Test Equipment, Grounding & Interfaces

By Rick Donaldson, NØNJY — n0njy@qsl.net

4.1 Introduction

This module covers the practical side of setting up and operating an HF station: transceiver controls new to HF operation, test equipment, interference reduction, digital interface hardware, and proper station grounding. These are everyday operating skills you will use constantly once on the HF bands.

4.2 HF Transceiver Controls

  • IF Filter: Controls the bandwidth of signals passed to the detector. Narrower filter rejects out-of-band interference. Typical SSB filter: 2.4 kHz; CW filter: as narrow as 250 Hz.
  • Notch Filter: Places a sharp null within the passband, eliminating a single interfering carrier without significantly affecting the desired signal.
  • Noise Blanker (NB): Suppresses pulse-type interference such as automotive ignition noise. Works by muting the receiver during brief, high-amplitude noise pulses.
  • DSP Noise Reduction: Can dramatically improve readability. However, aggressive settings introduce audio artifacts (“watery” sound).
  • AGC (Automatic Gain Control): Prevents strong signals from overdriving the receiver. Fast AGC useful for CW; slow AGC preferred for SSB.
  • RIT (Receiver Incremental Tuning): Adjusts receive frequency slightly without changing transmit frequency. Useful when the other station is slightly off frequency.

4.3 Test Equipment

SWR Meter: Measures impedance match between transmitter and antenna. Inserted in series with the feedline.

Dummy Load: A non-radiating resistive load for testing transmitters. All testing and adjusting should be done into a dummy load, not into an antenna.

Oscilloscope: Displays waveform shape versus time. Used to check modulation, identify distortion, and verify CW keying waveforms.

Spectrum Analyzer: Displays amplitude versus frequency. Use this to identify spurious emissions, check signal bandwidth, or identify interference sources.

Grid Dip Oscillator (Dip Meter): Determines the resonant frequency of an antenna or tuned circuit without connecting test leads.

4.4 Antenna Tuners

An antenna tuner (coupler) matches the impedance of the transmission line to the 50-ohm output of the transmitter. It does not change the antenna’s resonance. A tuner can make the transmitter happy while the antenna system still radiates poorly if the feedline has high loss at elevated SWR.

4.5 Station Grounding

Safety ground: Connects all station equipment chassis to the AC safety ground. Prevents dangerous voltages on equipment cases in the event of a fault.

RF ground: The return path for the antenna system. A poor RF ground reduces antenna efficiency and causes RF in the shack. Use a short, wide conductor (copper strap preferred over wire) from the station to a ground rod or radial field.

RF in the shack symptoms: Microphone feedback, erratic computer behavior during transmit, tingling from equipment surfaces. Cure: proper bonding and common-mode chokes on feedlines.

4.6 Digital Mode Interfaces

Connecting a computer for digital modes requires an interface handling audio in/out and PTT control. Proper interfaces include audio isolation transformers to prevent ground loops and RF feedback. CAT control allows software to control the transceiver’s frequency and mode. Most digital software (WSJT-X for FT8) requires CAT control to function properly.

Module 4 Practice Questions

Q1 (G4A01) — What is the purpose of the notch filter found on many HF transceivers?

  1. To reduce noise from local atmospheric sources
  2. To reduce interference from carriers in the receiver passband
  3. To reduce power supply hum in the receiver
  4. To remove impulse noise from the receiver

Q2 (G4B05) — What is an advantage of digital signal processing (DSP)?

  1. It eliminates all types of interference
  2. It can perform filtering and noise reduction difficult with analog circuits
  3. It increases the transmitter power output
  4. It increases the transceiver's frequency stability

Q3 (G4D01) — What is the purpose of a speech processor in an SSB transmitter?

  1. To reduce transmitter power consumption
  2. To improve voice quality during contest operation
  3. To increase intelligibility by increasing average power
  4. To remove distortion from the microphone signal

Q4 (G4B01) — Which test instrument is used to display the shape of a transmitted waveform?

  1. A spectrum analyzer
  2. A frequency counter
  3. An oscilloscope
  4. A field strength meter

Q5 (G4B02) — What is an advantage of a spectrum analyzer over an oscilloscope for RF measurements?

  1. It can measure voltages in AC circuits
  2. It can display signal amplitude versus frequency simultaneously
  3. It is more accurate for time-domain measurements
  4. It requires less calibration

Q6 (G4A12) — What is the function of automatic gain control (AGC)?

  1. It adjusts transmitter power based on signal strength
  2. It prevents strong signals from overdriving the receiver
  3. It controls the bandwidth of the IF filter
  4. It reduces the output of the final amplifier stage

Q7 (G4C05) — What is the most common cause of RF interference from an HF mobile installation?

  1. Inadequate bonding of the vehicle’s electrical system
  2. A poorly aimed antenna
  3. Excessive transmitter power
  4. A poorly filtered audio system

Q8 (G4E05) — Which of the following is a disadvantage of using a dummy load?

  1. It does not allow measurement of SWR
  2. It will not allow adjustment under actual operating conditions
  3. It cannot be connected to an antenna tuner
  4. It converts RF power to heat without radiating

Answer Key

  1. B — A notch filter eliminates a specific interfering carrier within the passband
  2. B — DSP can implement filtering and noise reduction functions difficult in analog
  3. C — A speech processor increases average power, improving intelligibility
  4. C — An oscilloscope displays waveform shape in the time domain
  5. B — A spectrum analyzer shows amplitude vs. frequency simultaneously
  6. B — AGC prevents receiver overload from strong signals
  7. A — Inadequate electrical bonding is the most common cause of mobile RF interference
  8. D — A dummy load converts RF to heat but does not radiate (the intentional “disadvantage” the question targets)
